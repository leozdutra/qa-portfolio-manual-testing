# Bug Reports

## BUG-001

Title:
Checkout allows purchase beyond stock quantity

Environment:
Web / Chrome / Windows

Steps to Reproduce:
1. Select a product
2. Enter quantity greater than stock
3. Proceed to checkout

Expected Result:
System blocks purchase and displays warning

Actual Result:
Purchase is completed

Severity:
Critical
